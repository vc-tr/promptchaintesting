## Legacy: n8n + OpenAI Prompt Chain Demo

This directory contains the original n8n-based prompt chain demo that was the
first iteration of this project. It is preserved for reference but is no longer
the primary surface of the repository.

To run the legacy demo, `cd` into this directory and follow the instructions in
`docker-compose.yml` / `start.sh`. You will need Docker and an OpenAI API key.
